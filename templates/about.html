About BloodPrint

Revolutionizing blood group identification using fingerprint analysis and deep learning.

BloodPrint is an innovative prototype that explores the correlation between fingerprint patterns and blood groups. Using a convolutional neural network trained on curated fingerprint datasets, the system provides probability-driven predictions intended for research, academic demonstrations, and rapid screening support.

Deep Learning Core

Transfer learning with TensorFlow/Keras leverages pre-trained feature extractors to capture dermatoglyphic minutiae.

Explainable Confidence

Top-three predictions with confidence percentages offer transparency into the model output.

Lab-Friendly Workflow

Simple upload experience that aligns with modern diagnostic dashboards for laboratory pilots.

Human Oversight

Designed as a decision-support tool; medical professionals must validate critical outcomes.

Model Snapshot
  • Architecture: CNN with transfer learning
  • Accuracy: ~91% validation
  • Classes: 8 blood groups
Responsible Usage

BloodPrint is intended as a research and educational demonstrator. It should not be used as a standalone medical diagnostic. Always corroborate predictions with laboratory-grade testing and clinical judgement.
